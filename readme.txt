== Description ==

🎯 Key Features Added:
Admin Settings Panel

Settings Location: WordPress Admin → Settings → GA4 Ecommerce
GA4 Code Management: Place your complete GA4 tracking code
Code Placement Options: Header, Footer, or After Body Tag
Event Toggle Controls: Enable/disable each tracking event individually

Event Controls

✅ View Item Event (product pages)
✅ Add to Cart Event (AJAX & regular)
✅ Begin Checkout Event (checkout page)
✅ Purchase Event (thank you page)

Original Code Preserved

All your existing tracking logic is identical
Same AJAX handling for add to cart
Same currency detection
Same product data collection
Same variation support

📦 Installation Instructions:

Upload: Place it in /wp-content/plugins/ga4-enhanced-ecommerce/ folder
Activate: Go to WordPress Admin → Plugins → Activate "GA4 Enhanced Ecommerce Tracking"
Configure: Go to Settings → GA4 Ecommerce to set up your tracking

⚙️ Configuration Steps:

Add GA4 Code: Paste your complete GA4 tracking code (including script tags)
Choose Placement: Select Header, Footer, or After Body Tag
Enable Events: Toggle individual events on/off as needed
Save Settings: Click "Save Settings"
